# CNBlogs 同步插件 - 快速开始

一个 WordPress 插件，用于将文章自动同步到 CNBlogs。

## 📋 文档

- **[用户指南](USER-GUIDE.md)** - 包含安装、配置及故障排查
- **[更新日志](CHANGELOG.md)** - 所有版本的详细修复说明
- **[文档索引](README.md)** - 文档导航

## ✨ 功能

- ✅ 自动同步文章到 CNBlogs
- ✅ 支持自动更新和删除
- ✅ MetaWeblog API 支持
- ✅ WordPress 6.9 兼容
- ✅ 简洁的后台界面
- ✅ 支持覆盖升级

## 🚀 快速使用

1. 上传插件到 `wp-content/plugins/cnblogs-sync/`
2. 在 WordPress 后台激活插件
3. 进入 **CNBlogs 同步** → **设置**
4. 输入你的 CNBlogs 用户名和令牌
5. 点击"测试连接"验证
6. 按需启用"自动同步"或"同步标签/分类"功能
